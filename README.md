# Standard Lesson Plan Skill

这是一个用于整理教案的 Codex Skill，主要面向学校“三段五学六有”课堂要求。它可以将原始教学设计或零散教案内容，重新整理为结构清晰、语言简约、适合直接放入学校“六有课堂”教案模板的 Word 文档。

## 功能

- 根据“三段五学六有”要求重组教案内容
- 自动梳理学情分析、教学目标、教学重难点、教学过程、当堂检测、作业巩固、教学反思等栏目
- 将教学过程整理为“三段”结构：
  - 情境导入
  - 探究建构
  - 迁移应用
- 在教学过程中融入“五学”：
  - 导学
  - 自学
  - 互学
  - 评学
  - 创学
- 强化“六有”要求：
  - 有学情分析
  - 有明确目标
  - 有方法路径
  - 有精准点拨
  - 有评测反馈
  - 有学力建设
- 参考固定六有课堂教案格式输出 Word
- 去除圆点项目符号，改用自然分行或 `1. 2. 3.` 编号
- 压缩口语化、重复、碎片化表达，使内容更符合正式教案语言

## 适用场景

适用于将已有的教学设计、课堂流程稿、说课稿等内容，整理成学校统一格式的六有课堂教案。

特别适合：

- 赛课教案格式调整
- 原始教案转“三段五学六有”结构
- Word 教案模板填充
- 多份教案批量规范化整理

## 输出特点

- 结构清晰
- 表述简洁
- 教案化语言
- 适合直接用于 Word 备课文档
- 保留原课题与核心教学内容
- 不随意编造课程主题
- 格式统一，便于后续多次调用
